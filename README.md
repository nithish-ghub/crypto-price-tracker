# crypto-price-tracker

A Python-based cryptocurrency tracking project that collects real-time crypto prices and market data using web scraping and automation tools.

## Features

* Tracks live cryptocurrency prices
* Displays market trends
* Stores scraped data in CSV format
* Automated browser scraping using Selenium

## Technologies Used

* Python
* Selenium
* Pandas
* WebDriver Manager

## How to Run

```bash
pip install -r requirements.txt
python main.py
```

## Output

The project generates real-time cryptocurrency tracking data and exports it into CSV format.
